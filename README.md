# Konvergensi-Metode-Iterasi-Titik-Tetap
Mini project metode numerik menggunakan Metode Iterasi Titik Tetap (Fixed Point Iteration) untuk menyelesaikan persamaan nonlinear f(x)=x 3 −x−2.  Dalam project ini dilakukan proses iterasi numerik menggunakan beberapa bentuk fungsi g(x) untuk menganalisis konvergensi menuju akar persamaan dengan bantuan Microsoft Excel.
